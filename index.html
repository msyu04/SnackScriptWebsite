About SnackScript

SnackScript is a spellbinding programming language that blends wizardry with food magic. Designed for readability and fun, it allows developers to cast spells with ingredients rather than traditional syntax.

With SnackScript, loops become "recipes," functions are "spells," and variables are "ingredients." If you've ever wanted to cook up some code, this is the language for you!

Example Programs


# Hello World in SnackScript
🥘 say_hello:
  🍦 This is a single line comment
  🍨 This is a
     multi-line comment
     :) 🍨
  🍽️ "Hello, world"
;
        

# Nested Loops
🍥 (🍳 i = 1, i <= 10, i++):
🍳 row = ""
🍥 (🍳 j = 1, j <= 10, j++):
  row += (i * j) + "\t"
;
🍽️ row
;
        

# Lists
🥡 student_scores = [ ("Annie", 91), ("Barbara", 58), ("Charlie", 49), ("Daniel", 51) ]
🥘 determine_grade(score):
  🧁 score >= 90:
    🫗 "A"
  ;
  🍰 score >= 80:
    🫗 "B"
  ;
  🍰 score >= 70:
    🫗 "C"
  ;
  🍰 score >= 60:
    🫗 "D"
  ;
  🎂:
    🫗 "F"
  ;
;
🍱 student_grades = {name: determine_grade(score) for name, score in student_scores}
🍥 student, grade in student_grades.🥚:
  🍽️ student + ":" + grade
;
        

# Multiplication Table
🍥 (🍳 i = 1, i <= 10, i++):
  🍳 row = ""
  🍥 (🍳 j = 1, j <= 10, j++):
    row += (i * j) + "\t"
  ;
  🍽️ row
;
        

# Fibonacci
🥘 fib (🍳 n) :
  🍳 a = 0, b = 1, f = 1
  🍥 (🍳 i = 2, i < n, i++):
    f = a + b
    a = b
    b = f
  🫗 f
  ;
;
